Class Projects

ECEN 5013 Final Project: HAM Radio Receiver

Apr 2018

Designed a radio receiver in the 10 meter band, including circuit design of all filters and oscillators. Created receiver schematic and did PCB layout in Altium with a 4-layer design.

ECEN 3360 Final Project: Temperature Chamber

Dec 2017

Built a temperature-controlled chamber using a Sparkfun temperature sensor communicating over I2C with the LCXpress board. Temperature was controlled using MOSFETs to turn on cooling and heating elements with an algorithm to keep the temperature at a set point. Temperature was reported and set over bluetooth.

ECEN 2270 Final Project: "Budget Roomba"

May 2017

Built a “budget Roomba” that mapped a quadrilateral-shaped room, using a combination of analog feedback loops and an Arduino for speed control and line-finding

ECEN 2020 Final Project: Snake Game

Dec 2016

Programmed the game "Snake" on the MSP432 board with BoosterPack. Wrote separate libraries in Code Composer Studio for snake functions, snake display functions, and input configuration. Configured ADC on the MSP432 to convert joystick inputs for two axes

ECEN 2350 Project: Number Guessing Game

Dec 2016

Designed and implemented a number guessing game on the Altera DE0 FPGA. Explanation video here.

Digital Clock Design Project

Nov 2015

Designed and built a digital 12-hour clock with AM/PM indicator and alarm using logic gates and counters. Designed and tested circuitry using Multisim before implementation. Built custom 5-Volt AC/DC power supply to power clock from the wall.
